# css-resolution-units

[![Build Status][ci-img]][ci]

List of CSS [`<resolution>`][resolution] units.

## Install

```sh
npm install css-resolution-units --save
```

## Usage

```js
var cssResolutionUnits = require('css-resolution-units');
console.log(cssResolutionUnits);
/* [
	"dpi",
	"dpcm",
	"dppx"
] */
```
